Build me a tablet-friendly vehicle dashboard web app for my 2010 Subaru Forester XS.

The app will run on a Samsung tablet mounted in the car and act as a complete vehicle information, maintenance, camping, and future vehicle control hub.

GENERAL REQUIREMENTS

- React
- Tailwind CSS
- Dark mode by default
- Subaru-inspired blue accent colours
- Mobile and tablet first design
- Offline capable
- Local storage for all data
- Import/export backup system using JSON
- Large buttons and text suitable for use in a vehicle
- Modern dashboard appearance
- Fast loading
- Easy to expand in the future
